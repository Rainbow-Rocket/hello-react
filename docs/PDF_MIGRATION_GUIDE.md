# PDF 迁移指南

## 第一步：将 Word 文档转换为 PDF

### 方法1：使用 Microsoft Word（推荐）

#### macOS 版本步骤：

1. 打开 Word 文档
2. 点击 **文件** → **另存为**
3. 选择保存位置：`public/documents/news/`
4. 在 **文件格式** 下拉菜单中选择 **PDF**
5. **优化文件大小（macOS）**：
   - 在"另存为"对话框中，点击 **选项...** 按钮
   - 在弹出窗口中：
     - 勾选 **最小文件大小（联机发布）**
     - 图像质量选择 **最低** 或 **低**
   - 点击 **确定**
6. 点击 **保存**

#### 如果 Word 中没有"选项"按钮：

**方法A：先压缩图片再转 PDF**
1. 在 Word 中，选中任意一张图片
2. 在顶部菜单栏会出现 **图片格式** 选项卡
3. 点击 **图片格式** → **压缩图片**
4. 选择 **Web 质量** 或 **低质量**
5. 勾选 **应用于文档中的所有图片**
6. 点击 **确定**
7. 然后再执行"另存为 PDF"操作

**方法B：直接保存，后续压缩 PDF**
1. 直接另存为 PDF（不压缩）
2. 使用在线工具压缩 PDF（见下方"压缩 PDF 文件"部分）

### 方法2：批量转换（如果有多个文件）

**使用 macOS 的自动操作：**
1. 打开"自动操作"应用
2. 创建新的"工作流程"
3. 添加"查找 Finder 项目"操作
4. 添加"将 Finder 项目转换为 PDF"操作
5. 运行工作流程

**或使用在线工具：**
- https://www.ilovepdf.com/word-to-pdf
- https://smallpdf.com/word-to-pdf

## 第二步：文件命名

PDF 文件名应该与 Word 文件名对应，例如：

```
Word:  1、佛罗伦萨大学学生生涯实验室中国中心在沪正式成立：集大学准备、科研、实习、国际交流和其他与大学相关的活动于一体.docx
PDF:   1、佛罗伦萨大学学生生涯实验室中国中心在沪正式成立：集大学准备、科研、实习、国际交流和其他与大学相关的活动于一体.pdf
```

## 第三步：更新 Markdown 文件

将所有的 `<WordViewer>` 替换为 `<Viewer>`，并修改文件扩展名：

**修改前：**
```markdown
<WordViewer docUrl='/documents/news/1、xxx.docx' />
```

**修改后：**
```markdown
<Viewer uri="/documents/news/1、xxx.pdf" />
```

## 第四步：验证

1. 启动开发服务器：`yarn dev`
2. 访问新闻页面，检查 PDF 是否正常加载
3. 测试滚动加载功能（应该只加载前几页）

## 优化建议

### 压缩 PDF 文件（可选，但推荐）

转换后，可以使用以下工具压缩 PDF：

**macOS 命令行（如果安装了 ghostscript）：**
```bash
cd public/documents/news/
for file in *.pdf; do
  gs -sDEVICE=pdfwrite -dCompatibilityLevel=1.4 \
     -dPDFSETTINGS=/screen -dNOPAUSE -dQUIET -dBATCH \
     -sOutputFile="${file%.pdf}_compressed.pdf" "$file"
  mv "${file%.pdf}_compressed.pdf" "$file"
done
```

**在线工具：**
- https://www.ilovepdf.com/compress-pdf
- https://smallpdf.com/compress-pdf

压缩通常可以减少 30-50% 的文件大小，显著提高加载速度。
